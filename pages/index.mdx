# Welcome to Hooked on React

A collection of handy React hooks designed to accelerate development and enrich the user experience with less code and more functionality.

## Why Hooked on React?

This library is crafted to address common React patterns and challenges. By abstracting these patterns into reusable hooks, you can keep your components clean and focus on the unique parts of your application.

## Features

- **Performance Optimised:** Hooks are optimised to prevent unnecessary renders.
- **Type-Safe:** Fully typed with TypeScript for a better developer experience.
- **Community-Driven:** Open for contributions and improvements from the community.

## Quick Start

Get started with `hooked-on-react` in your project by installing the package:

```bash
npm install hooked-on-react
# or
yarn add hooked-on-react
```
